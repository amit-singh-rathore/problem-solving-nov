# 414. Third Maximum Number

Given integer array nums, return the third maximum number in this array. If the third maximum does not exist, return the maximum number.

## Example

```
Input: nums = [3,2,1]
Output: 1
Explanation: The third maximum is 1.

```

## Constraints

- 1 <= nums.length <= 10^4
- -2^31 <= nums[i] <= 2^31 - 1
