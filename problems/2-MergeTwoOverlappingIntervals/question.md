# Arrays: Merge two overlapping intervals

Given an array of interval pairs, sorted with interval start, merge overlapping intervals and return new array

## Example:
*Input:* [(1, 5), (3, 7), (4, 6), (6, 8)] 
*Output:* [(1, 8)]

*Input:* [(10, 12), (12, 15)] 
*Output:* [(10, 15)]

*Input:* [(1, 5), (7, 9), (8, 9), (9, 12)] 
*Output:* [(1, 5), (7, 12)]
